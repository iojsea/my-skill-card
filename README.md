# my-skill-card

个人技能卡片网页，我的第一个 GitHub 项目。

## 技术栈

- HTML
- CSS

## 本地运行

1. 克隆仓库
   ```bash
   git clone https://github.com/iojsea/my-skill-card.git
   ```
2. 用浏览器打开 `index.html`

## 项目结构

```
my-skill-card/
├── index.html    # 技能卡片页面
└── README.md     # 项目说明
```

## 学习笔记

本项目跟随 Marvis 的 GitHub 教学案例一步步搭建，完整经历了：

- 建仓库 → 克隆 → 编码 → 提交
- 分支开发 → Pull Request → 合并
